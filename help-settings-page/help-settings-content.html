Emoji Translator

You can translate text into emoji using this addon. By default typing the emoji name specified on the 'dictionary' tab in between ':' i.e :heart: to ❤️. You can change the starting and end character to whatever you find convenient. Also the order of apperance of favourites emoji icons can be changed in the 'settings' tab through simple drag and drop.
